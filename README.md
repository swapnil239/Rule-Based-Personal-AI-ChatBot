# 🤖 Rule-Based Basic AI Chatbot (Python)

A beginner-friendly Rule-Based Chatbot built using Python.
This chatbot responds to basic user queries using predefined rules and dictionary-based responses.

---

## 📌 Features

* 👋 Greets user based on current time
* 🧠 Dictionary-based memory system
* 💬 Responds to basic questions
* 🔁 Continuous chat loop
* 🚪 Exit option using `goodbye`
* 🔡 Case-insensitive input handling

---

## 🛠 Technologies Used

* Python
* Dictionary (Data Structure)
* Functions
* Loops
* Conditional Statements
* datetime module

---

## 📂 How It Works

1. User enters their name
2. Bot greets according to current time
3. User asks a question
4. Bot searches in predefined response dictionary
5. If match found → returns answer
6. If no match → default learning message
